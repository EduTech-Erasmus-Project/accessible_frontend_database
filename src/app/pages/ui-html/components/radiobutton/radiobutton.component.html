Botón de radio

type = "radio"

Descripción

Representa una opción en donde no existe más de una opción seleccionada al mismo tiempo. Estos grupos común mente están conformados por un numero de botones de opción, todos comparten el mismo valor en el atributo name.

Dentro de las casillas de verificación, el valor del atributo value tiene un papel importante para los botones de opción.

Etiqueta HTML :

<input type="radio">

Codigo HTML



<label>Seleccione un numero par:</label>

</div>
