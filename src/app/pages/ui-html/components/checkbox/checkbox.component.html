Checkbox

type = "checkbox"

Descripción

Es un control de dos estados que permiten al usuario marcarlo como seleccionado o deseleccionado. Puede ser usado para recolectar información, como por ejemplo responder preguntas de estado verdadero/falso como “¿correr/caminar/saltar?”

El elemento <input>, tiene un valor “checkbox” en su atributo type.

Etiqueta HTML :

<input type="checkbox">

Codigo HTML



<label>¿Cuales son tus colores favoritos?</label>
<label><input type="checkbox" id="cbox1" value="checkbox1"> Azul</label>
<label><input type="checkbox" id="cbox2" value="checkbox2"> Rojo</label>
<label><input type="checkbox" id="cbox3" value="checkbox2"> Amarillo</label>
