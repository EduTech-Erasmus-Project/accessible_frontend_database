Calendario

calendar

Descripción

El calendario en HTML se describe con los elementos de la siguiente manera <input> de type="date" presenta una interfaz especial que le permite escoger una fecha o un campo de texto que le permite ingresar la fecha de manera textual.

Su valor resultante incluye el año, el mes, el día.

Etiqueta HTML :

<input type="date">

Codigo HTML


<label for="start">Ingresar la fecha:</label>
<input type="date" id="start" name="trip-start" value="2018-07-22" min="2018-01-01" max="2018-12-31">
