Lista desplegable

select

Descripción

Representa un control que permite al usuario elegir una opción de entre una lista de opciones o un numero de opciones. Las opciones contenidas en el menú son representadas por elementos {{option}}, los mismos que pueden ser agrupados por {{optgroup}}.

Etiqueta HTML :

<select>
  <option></option>
</select>

Codigo HTML


<select name="select">
  <option value="value1">Cañar</option>
  <option value="value2" >Cuenca</option>
  <option value="value3">Guayaquil</option>
</select>





<label for="provinciaI">Elige tu provincia favorita?</label>
<select id="provinciaI" name="select" style="width:50%">
<optgroup label="Sierra">
  <option value="value1">Cañar</option>
  <option value="value2">Cuenca</option> </optgroup>
<optgroup label="Costa">
  <option value="value3">Guayaquil</option>
</optgroup> </select>
