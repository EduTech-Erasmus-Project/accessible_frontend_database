• Boton

    Un botón es un widget que permite a los usuarios activar una acción o un evento, como enviar un formulario, abrir un cuadro de diálogo, cancelar una acción o realizar una operación de eliminación.

  • Select-Only Combobox

    Un cuadro combinado es un widget de entrada con una ventana emergente asociada que permite a los usuarios seleccionar un valor para el cuadro combinado de una colección de valores posibles.

  • Accordion

    Un acordeón es un conjunto de encabezados interactivos apilados verticalmente que contienen un título, fragmento de contenido o miniatura que representa una sección de contenido.

  • Calendario

    Una fecha de entrada de fecha editable Combobox que abre un diálogo que contiene una cuadrícula y botones de calendario para navegar por mes y año.

  • Checkbox

    WAI-ARIA admite dos tipos de widgets de casilla de verificación: el estado dual permite al usuario alternar entre dos opciones (marcado y no marcado) y el estado triple admite un tercer estado adicional conocido como marcado parcialmente.

  • Radio Button

    Un grupo de opción es un conjunto de botones de casilla de verificación, conocidos como botones de opción, donde no se puede marcar más de uno de los botones a la vez.
