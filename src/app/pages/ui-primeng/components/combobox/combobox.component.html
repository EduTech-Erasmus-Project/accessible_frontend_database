Ejemplo de Combobox en ARIA (APG)

Descripción

Un cuadro combinado es un widget de entrada con una ventana emergente asociada que permite a los usuarios seleccionar un valor para el cuadro combinado de una colección de valores posibles.

Etiqueta HTML :

<p-dropdown></p-dropdown>

Codigo HTML

<p-dropdown [options]="cities" [(ngModel)]="selectedCity1" placeholder="Select a City" optionLabel="name" [showClear]="true"></p-dropdown>
