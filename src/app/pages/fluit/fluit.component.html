Intregrar UI Options con un proyecto de Angular

1. Primero nos descargamos el codigo fuente del repositorio de GitHub de fluid-project: https://github.com/fluid-project/infusion/archive/master.zip

2. Descomprima el archivo .zip

3. Mueva la carpeta resultante infusion a algún lugar conveniente para sus propósitos de desarrollo. En este caso moveremos a la carpeta de assets.

4. El panel deslizante

Es una pequeña plantilla de HTML para todos los controles, por lo que no es necesario crear ningun HTML.

5. Agregar dependencias a la página

Las opciones de la interfaz de usuario dependen del sistema Fluid Skinning (FSS) y de Infusion Framework, por lo que deberá agregar dependencias a sus páginas.

En el encabezado de su archivo, enlace a los archivos FSS CSS con etiquetas (es posible que deba ajustar las rutas para reflejar dónde guardó el paquete de Infusion).

6. Agregue el componente de opciones de interfaz de usuario

La forma más sencilla de agregar el componente Opciones de IU a su página es usar una <script> etiqueta cerca de la parte superior de la página. Sugerimos colocarlo justo antes del marcado de Opciones de interfaz de usuario creado en el Paso 1 .
