Componentes Accesibles

Rumbo a una Web accesible

La visión de este proyecto es lograr una integracion de diferentes componentes con la finalidad de tener una Web accesible, para aumentar la usabilidad de la Web, eliminando barreras que dificulten el acceso a la Web para todo tipo de usuarios mejoran el contenido dentro de las plataformas web.


Componentes HTML

En este apartado de la pagina web, ver los componentes con su especificación realizados con los estándares de HMTL lo más sencillos posibles. Tratamos de mostrar cada componente con su estructura adecuada para luego mostrar los cambios necesarios para llegar a obtener los componentes accesibles.

Componentes con ARIA

Accessible Rich Internet Application

Los componentes de ARIA, tiene una semántica accesible para crear experiencias en el web de manera accesible. Tiene una guía que describe como aplicar esta semántica de accesibilidad a widgets y patrones de diseño comunes.
